# Backend Modules Structure

- **auth**: JWT, session, device logic
- **user**: Profile, progress, analytics
- **course**: Catalog, lessons, progress
- **test**: Mock tests, scoring, analytics
- **quiz**: Daily quiz, leaderboard
- **admin**: User, course, test management
- **payment**: Stripe/Razorpay integration
- **notification**: Push, reminders
- **engagement**: Gamification, referrals
- **ai**: Chatbot, recommendations
